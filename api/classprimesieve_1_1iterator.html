Detailed Description

Iterate over prime numbers.

The primesieve_iterator.cpp and previous_prime.cpp examples show how to use primesieve::iterator objects.

Note
primesieve::iterator objects are very convenient to use at the cost of being slightly slower than the callback_primes() functions.
Examples:
previous_prime.cpp, and primesieve_iterator.cpp.
