Detailed Description

Fast prime iterator for use in C code.

The primesieve_iterator.c example shows how to use primesieve_iterator. If any error occurs errno is is set to EDOM and primesieve_next_prime() and primesieve_previous_prime() return PRIMESIEVE_ERROR (UINT64_MAX).
